# prep-javascript-conditionals-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What does `strictly equal` mean?
  It is a comparison operator that returns a boolean, It only returns true if it is the exact same value and data type of the thing it is comparing.
- What is the logical **and** operator?
  it is '&&' and you use it when you want to check multiple conditions are met.
- Can you name some comparison operators?
  strictly equals = '===', greater than = '>', less than '<', greater than or equals to = '>=', less than or equals to '<=', not equals to = '!=='
